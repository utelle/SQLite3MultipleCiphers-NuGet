This package contains the native SQLite3 Multiple Ciphers binaries
for all supported platforms.️ It is a low-level package.

It is intended for advanced scenarios where you need full control
over the integration.

Most users should use [SQLite3MC.PCLRaw.bundle](https://www.nuget.org/packages/SQLite3MC.PCLRaw.bundle) instead.

More detailed information can be found in the [_SQLite3MC.PCLRaw.bundle's_ README](https://www.nuget.org/packages/SQLite3MC.PCLRaw.bundle#readme-body-tab).
