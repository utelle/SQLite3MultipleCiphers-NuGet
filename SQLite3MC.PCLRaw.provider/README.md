This package provides the integration layer for SQLitePCLRaw. It is part of the `SQLite3MC.PCLRaw` stack.

Use this package only if you need custom configuration or want to combine it with your own native library setup.

Most users should use [SQLite3MC.PCLRaw.bundle](https://www.nuget.org/packages/SQLite3MC.PCLRaw.bundle).

More detailed information can be found in the [_SQLite3MC.PCLRaw.bundle's_ README](https://www.nuget.org/packages/SQLite3MC.PCLRaw.bundle#readme-body-tab).
